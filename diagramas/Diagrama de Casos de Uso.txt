	UC1- Acessar o mapa

	1 - O jogador abre o jogo
	2 - O jogador entra na tela inicial do jogo
	3 - O jogador clica em "Jogar"
	4 - Incluir UC2 (Adicionar jogadores ao jogo)
	5 - O jogador visualiza o mapa do jogo


	UC2 - Adicionar jogadores

	1 - O sistema abre a tela de adição de jogadores
	2 - O jogador digita um nome
	3 - O jogador escolhe a opção "Humano" ou "Computador"
	4 - O jogador seleciona a casa de sua preferencia
	5 - O jogador clica em "Adicionar jogador"
	6 - O sistema valida o jogador
	7 - O jogador clica em "Jogar"
	8 - O sistema verifica o num de jogadores

		5.a - Atingiu o máximo de jogadores para a partida
			1 - O sistema desabilita o botão "Adicionar jogador"
			2- Avança ao passo 7

		6.a - O jogador não digita um nome
			1 - O sistema exibe uma mensagem "Digite o seu nome"
			2 - Volta ao passo 2

		6.b - O jogador não escolhe uma casa
			1 - O sistema exibe uma mensagem "Selecione a casa que deseja representar"
			2 - Volta ao passo 3

		7.a - O jogador deseja adicionar outro jogador
			1 - O jogador clica em "Adicionar Jogador"
			2 - Volta ao passo 2

	
	UCX - Acessar opções de jogo

	UCX - Acessar créditos do jogo

	UCX - Sair do jogo


	UC3 - Participar da rodada

	1 - O jogador recebe os exércitos a que tem direito do sistema
	2 - Incluir UC5 (Distribuir exércitos pelos territorios)
	3 - Incluir UC4 (Atacar o território)
	4 - Incluir UC6 (Deslocar exércitos)
	5 - O jogador escolhe entre passar a vez ou continuar na rodada
	
		4.a - O jogador da vez perdeu a batalha
			1 - Avança ao passo 5

		5.a - O jogador passa a vez
			1 - Estendido por UC8
			2 - O sistema informa ao jogador "A sua jogada terminou"

		5.b - O jogador continua na rodada
			1 - Volta ao passo 2

		5.c - O jogador consegue conquistar o objetivo da missão
			1 - O sistema declara o jogador como vencedor
			2 - Fim de jogo


	UC4 - Atacar

	1 - O jogador clica em "Atacar"
	2 - O sistema mostra os territórios que podem ser selecionado para realizar o ataque 
	2 - O jogador seleciona um territorio de sua posse no mapa
	3 - O jogador seleciona um território que quer atacar
	4 - O jogador digita o número de exércitos para o ataque no campo "Número de exércitos para ataque"
	5 - O jogador atacado digita o número de exércitos para defender no campo "Número de exércitos para a defesa"
	6 - O jogador clica em "OK"
	7 - O sistema executa o sorteio dos dados
	8 - O sistema avalia quem venceu a batalha
	9 - O sistema remove os exércitos perdedores

		2.a - O jogador tem apenas um exército no território selecionado
			1 - O sistema informa "Voce não pode atacar apenas com o exercito de ocupação"

		3.a - O jogador escolhe um território não vizinho ao territorio atacante
			1 - O sistema informa "Escolha um territorio vizinho para o ataque"
	
		4.a - O jogador escolhe um número acima do que possui
			1 - O sistema informa "Você não tem esse numero de exercitos"
	
		4.b - O jogador digita um numero maior que 3
			1 - O sistema informa "Você só pode atacar com no máximo 3 exercitos"

		5.a - O jogador escolhe um número acima do que possui
			1 - O sistema informa "Você não tem esse numero de exercitos"
	
		5.b - O jogador digita um numero maior que 3
			1 - O sistema informa "Você só pode defender com no máximo 3 exercitos"


	UC5 - Distribuir exércitos pelos territorios

	1 - O jogador clica em "Rearrumar exércitos"
	2 - O jogador seleciona um território de sua propriedade no mapa
	3 - O jogador coloca um número de exércitos no território

		3.a - O jogador não tem exércitos suficientes
			1 - O sistema informa "Você não tem exercitos suficientes para distribuir pelos territórios"

	PS: territorios tem limites de execito????
		

	UC6 - Deslocar exércitos

	1 - O jogador clica em "Deslocar exércitos"
	2 - O jogador digita numero de exercitos a serem deslocados

		2.a - O jogador não deixa pelo menos um exercito no territorio atacante
			1 - O sistema informa "Deixe no minino 1 exercito para a ocupação"

	UC7 - Visualizar tabelas

	1 - O jogador abre o sistema
	2 - O jogador entra na tela inicial do jogo
	3 - O jogador clica em "Opções"
	4 - O jogador seleciona "Visualizar tabela de territórios"
	5 - O sistema mostra a tabela requisitada

		4.a - O jogador deseja visualizar a tabela de bônus
			1 - O jogador seleciona "Visualizar tabela de bônus"

	UC8 - Trocar cartas

	1 - O jogador clica em "Trocar cartas"
	2 - O jogador seleciona as cartas a serem trocadas
	3 - O sistema valida a troca
	4 - O jogador recebe os exercitos a que tem direito
	5 - Incluir UC5 (Distribuir exércitos pelos territorios)

		3.a - O jogador não tem cartas suficientes
			1 - O sistema informa "Você não tem cartas suficientes para fazer a troca"

		3.b - A combinação de cartas está errada
			1 - O sistema informa "A combinação de cartas está errada"
			2 - Volta ao passo 1


	UC9 - Definir início de jogo

	1 - O sistema sorteia a ordem do jogo
	2 - O jogador visualiza a sua posição na lista de jogadores da tela de jogo
	3 - O sistema distribui as cartas de território para os jogadores
	4 - O jogador clica em "Cartas"
	5 - O jogador visualiza a carta(s) recebida(s)
	6 - O sistema distribui o objetivo/ missão aos jogadores
	7 - O jogador clica em "Ver objetivo"
	8 - O jogador vê o objetivo
	9 - O jogador recebe os exécitos do sistema
	10 - O jogador, na sua vez, seleciona um territorio a sua escolha
	11 - O jogador coloca um número de exércitos no território

		11.a - O jogador não tem exércitos suficientes
			1 - O sistema informa "Você não tem exercitos suficientes para colocar neste território"
